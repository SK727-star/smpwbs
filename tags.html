重要なHTMLタグ一覧

このページは、初心者シリーズWeb基礎編 「HTML」で多用されるタグのいくつかをすぐ参照できるクイックリファレンスページ として作られます。このページ製作を通じて、HTMLのテーブル（table）をtable タグやそれに関連するタグの使い方を学びます。

以下のテーブルに含まれるHTMLタグは全体の一部に過ぎませんが、重要なHTMLついてはここで参照できます。

ブロック要素（block elemnt）

HTMLタグ タグ名 目的
h1–h6 見出し（heading） この中に見出しを置く（レベル: 1–6）
p 段落（パラグラフ: paragraph） この中に文章を入れる
table 表（テーブル） この中にテーブルを置く
tr 表の行（table row） この中に行データを置く
th 表の見出し（table header） この中に表の見出しを置く
td 表のデータ（table data） この中に表のセルデータを置く
div, header コンテンツ区分（division, header） コンテンツをページ内で区切る際に使う。divとheaderタグの実質的な機能は同じ
ul 順序なしリスト要素 項目に順序を与えずにリストを表示します
ol 順序ありリスト要素 項目に順序を与えて（連番で）リストを表示します。
li 項目要素 <ul>や、<ol>の子要素として使われる。リストの一単位。
HTMLタグ タグ名 目的 記述例 画面上の表示結果
em 軽い強調（emphasized） テキストを軽く強調する 技術の成熟（<em>technical sophistication</em>） 技術の成熟（technical sophistication）
strong 強調（strong） テキストを強調する <strong>少なくとも10億人</strong> 少なくとも10億人
a アンカー（anchor） ハイパーリンクにする <a href="https://learnenough.com/">Learn enough</a> Learn enough
img 画像（image） 画像を配置する <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code ソースコード（code） 文字列を等幅で表示する <code>Class Foo < Bar</code> Class Foo < Bar
span （インラインの）区分け（span） インラインで区画を分ける <span>こちら</span>です こちらです。（スタイルを指定しないと見かけ上は変化しない）
